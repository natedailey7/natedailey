Game Projects

During spring 2022, I was in a game engine programming class at Pomona College. Most game design takes place in pre-built engines, which deal with low-level details such as basic physics, lighting, collisions, etc. (such as the Unity or Unreal engines). However, in this class we were assigned to create a 2d-graphics engine and game, and a 3d game based on the bare-bones Frenderer engine, created by Professor Joseph Osborn. Following are samples of both games, Triumvirate Arena and JumpyBall.
